Accessibility Statement

This website is designed to be used by everyone and is made to be as accessible as possible, in accordance with the W3C Web Content Accessibility Guidelines.
By following these guideline the content of this website is accommodated for people with disabilities such as blindness, low vision, cognitive limitations and motor disabilities.

  • The website can be scaled up to 300% without losing its functionality
  • The colours are contrasting with readable font-size.
  • The website is suitable for keyboard use.
